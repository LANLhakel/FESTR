Finds common electron number density that is consistent with the charge-neutrality constraint for given temperature and stoichiometry of the Zone.

Parameters
[in]tblTable of materials
[in]te_inElectron temperature in the Zone
[in,out]iteElectron temperature Database index
[in]tr_inRadiation temperature in the Zone
[in,out]itrRadiation temperature Database index
[in]npTotal particle number density (particles/cm3)
[in]nmatNumber of materials in the Zone
[in]matList of materials in the Zone
[in]fpFractional populations of materials
[in,out]ineElectron number density Database index
Returns
Pair:
first: actual (not Table) electron number density in electrons/cm3,
second: string containing te, tr, ne labels for Database file access
