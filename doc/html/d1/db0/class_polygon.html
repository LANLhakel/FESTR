Detailed Description

Convex Polygons (derived from class Face)

The number of Node objects defining a Polygon is arbitrary. The Node locations are given via Vector3d objects, which are assumed to be coplanar.
Some methods in class Polygon (Polygon::area2_normal_center, Polygon::contains) are limited in their applicability to convex Polygons; their generalization to non-convex Polygons is currently left as a future enhancement.
