Detailed Description

Conical ribbons for 2-D RZ geometry (derived from class Face)

Each Cone consists of two Node objects, whose order defines the orientation of the Cone. In the Vector3d components of each Node the x coordinate represents R, and the y coordinate represents Z. The z coordinates of Nodes' Vector3d's are set to zero, and are ignored in class Cone.
